##READ ME

To start the program, you need to go in the package pubmanagement then execute the file : Management.java

To make the program work withour errors, you need to activate the preview feature of the pattern matching instance of if it is not already enabled (disabled by default) 
(in VM Options of the Run Configuration : --enable-preview)

All actions happen "turn by turn" with menu navigation where command lines are specified with [.]

It is possible to add elements in the ressources text files if you wish.
